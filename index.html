ORAC Submission Checker

This is a tool to check submissions from the site orac2.info. The problem id is the number that is at the end of the URL of orac problems.
Sorry that I haven't added any CSS yet so this looks ugly. Also, this is quite buggy because sometimes the CORS Proxy is down.
